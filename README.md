# MyCpu

### Running tests
Please make sure to create a libs folder inside your MyCpu project folder.
Download the following .jars and copy them to the libs folder:
- [junit-jupiter-api](https://repo1.maven.org/maven2/org/junit/jupiter/junit-jupiter-api) *(tested with v6.0.3)*
- [junit-jupiter-engine](https://repo1.maven.org/maven2/org/junit/jupiter/junit-jupiter-engine) *(tested with v6.0.3)*
- [junit-jupiter-params](https://repo1.maven.org/maven2/org/junit/jupiter/junit-jupiter-params) *(tested with v6.0.3)*
- [junit-platform-commons](https://repo1.maven.org/maven2/org/junit/platform/junit-platform-commons) *(tested with v6.0.3)*
- [junit platform-engine](https://repo1.maven.org/maven2/org/junit/platform/junit-platform-engine) *(tested with v6.0.3)*
- [opentest4j](https://repo1.maven.org/maven2/org/opentest4j/opentest4j) *(tested with v6.0.3)*
